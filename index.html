Qu'est ce qu'un maquetage Numérique

Il s’agit d’un aperçu, descriptif et visuel, d’un site ,service web ou d'une application. Il est la plus part du temps accompagné avec un «prototype»,qui lui est fonctionnel.

  • See more

Les Primordiaux: Le FED

Dans une application, les 3 points critiques sont les suivants:

  • Le Fonctionnel:Les idées, les fonctions, inscription,...
  • Design: Le Logo, les couleurs, les composants, les boutons, donner envie
  • L'Ergonomie:Interface Homme-Machine: Il permet de guider intuitivement les utilisateurs

Les Guidelines

Lignes directives permettant à l'utilisateur de se familiariser avec les systèmes d'applications:
style graphique (materiel design, flat design) , composant(navigation bar, Tab Bar)

Remarque:
